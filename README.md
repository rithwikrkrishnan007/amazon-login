### Basic commands 
 
- clone repo      : ```git clone <repo url>```

- initialise repo : ``` git init ```

- check status    : ``` git status ```

#### Basic Operations (initial commands) :

```
- git status

- git add file_name
- git add . (all files will add)

- git commit -m "message"

- git log 

- git push

```

No.|Command | Used for | 
------| -------| -------|
1| git clone | download existing remote repo.
2| git init | to initialize new git repo. (only use if the remote repo is empty) 
3| git status | to check the status
4| git add | add file to staging are
5| git commit -m "message" | files go to local repo with a message
6| git log | to check commits (logs)
7| git push | push commits to remote repo(from local repo)
8| git pull | take the changes from remote to local
9| git branch | check the current branch
10| git chekout branch_name | to switch between branches
11| git checkout -b origin branch_name | to switch and create new branch
12| git push --set-upstream origin branch_name | to push the branch into remote repo
